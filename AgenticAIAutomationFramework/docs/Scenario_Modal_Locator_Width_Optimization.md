# Scenario Details Modal - Locator Column Width Optimization

## ? Optimization Complete

### ?? **What Was Changed**

The Locator and Value columns in the Scenario Details modal have been optimized to display text more efficiently with automatic word-wrapping at approximately 15 characters per line.

### ?? **Changes Summary**

#### **Before:**
- Locator column: Auto width, no wrapping
- Value column: Auto width, no wrapping
- Long locators displayed on single line causing horizontal overflow
- Font size: 12px

#### **After:**
- **Table Layout**: Changed to `table-layout: fixed` for consistent column widths
- **Locator Column**: Fixed width of 150px (~15 characters at 11px font)
- **Value Column**: Fixed width of 120px (~12 characters at 11px font)
- **Action Column**: Reduced to 100px
- **Font Size**: Reduced to 11px for better fit
- **Word Wrapping**: Added `word-wrap: break-word`, `word-break: break-all`, `overflow-wrap: break-word`
- **Line Height**: Set to 1.4 for better readability of wrapped text

### ?? **New Column Widths**

| Column | Width | Purpose |
|--------|-------|---------|
| **#** | 50px | Step number |
| **Action** | 100px | Action type badge |
| **Locator** | 150px | Element locator (~15 chars/line) |
| **Value** | 120px | Action value (~12 chars/line) |
| **Actions** | 200px | Action buttons |

### ?? **Technical Implementation**

#### CSS Properties Added:
```css
/* Table Layout */
table-layout: fixed;  /* Enforce fixed column widths */

/* Locator & Value Columns */
word-wrap: break-word;      /* Break long words */
word-break: break-all;      /* Break at any character */
overflow-wrap: break-word;  /* Modern word wrapping */
line-height: 1.4;           /* Better spacing for wrapped text */
font-size: 11px;            /* Smaller font for more characters */
```
